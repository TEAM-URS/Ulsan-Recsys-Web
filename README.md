## 1. Project Abstract

### 프로젝트 개요

* 목적
    * 딥러닝 기반 개인화 추천시스템을 통해 울산의 다양한 음식과 관광지 경험에 도움을 줌

* 주요 기능
    * 딥러닝 기반 맛집/명소 추천
    * 구축된 DB를 통한 개인화 추천
    * 리뷰 및 평점 등록
